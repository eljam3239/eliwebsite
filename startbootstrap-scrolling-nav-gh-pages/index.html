Projects

My QMIND team is working on a voiceprint recognition program with the aim to improve voice assistants' ability to differntiate between speakers, or appropriately identifying speakers even if they're sick, for example.

This site is a basic project built with CSS, JavaScript and HTML. The UI and smooth scrolling features are implemented with Bootstrap and JQuery easing.

At a hackathon, I built a machine learning program using ScikitLearn that learned to accurately identify different kinds of clothing over 90% of the time (on tested data and on clothing images from Google Images).

I am most experienced with Python, where I have built a car database system, a text attitudonal-impact calculator which determines the "impact" of different components of language when given text and a program which determines the similarity in music tastes between 2 people.

These days, I am delving into competitive programming to improve upon my C/C++ skills that I gained from a course last fall, working on a full-stack project to continue learning about node.js, React and PostgreSQL, picking up the fundamentals of data science programming with Python and prepping for the hackathons at Queen's and U of T coming up in the next few months.
